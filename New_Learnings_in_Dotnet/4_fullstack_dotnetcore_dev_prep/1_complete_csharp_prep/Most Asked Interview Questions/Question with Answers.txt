Q1. What is C# and what are its main features?

Answer: C# is a modern, object-oriented programming language developed by Microsoft as part of the .NET initiative.
C# is used with the .NET framework for creating websites, applications, and games. 

C# has a varied reasons for being popular:-
1. Comparatively easier: Starting with C# is termed comparatively easier than other programming languages
2. Wider use of development: Using C#, you are prone to create web applications or gaming apps. C# has some fantastic features like automatic garbage collection, interfaces, etc. which help build better applications.
3. Larger target audience: Collaboration with Microsoft provides an edge to the applications created using C# since it would have a wider target audience.
4. It is one of the most popular programming languages in the world also easy to learn and simple to use.
5. It has huge community support
6. C# is an object-oriented language which gives a clear structure to programs and allows code to be reused, lowering development costs
7. As C# is close to C, C++ and Java, it makes it easy for programmers to switch to C# or vice versa

Main Features:-
1. Type-safe and object-oriented
2. Component-oriented programming
3. Automatic garbage collection
4. Cross-platform support via .NET Core/.NET
5. Strong integration with the .NET ecosystem
6. Support for LINQ, async/await, and generics
